                @Override
                public T getEntry(int givenPosition) {
                  T result = null;
              
                  if ((givenPosition >= 1) && (givenPosition <= numberOfEntries)) {
                    Node currentNode = firstNode;
                    for (int i = 0; i < givenPosition - 1; ++i) {
                      currentNode = currentNode.next;		// advance currentNode to next node
                    }
                    result = currentNode.data;	// currentNode is pointing to the node at givenPosition
                  }
              
                  return result;

IOT Project (Smart Ligthing)

I have used the light sensor, motion sensor, and the LED to build my module. The light sensor and motion sensor are the input devices, and the LED is the output device. My module has three modes: auto mode, manual mode, and motion mode. In auto mode, the LED turns on automatically when the surrounding light level is lower than 700, and motion is detected. The LED brightness adjusts automatically according to the surrounding light level. In manual mode, the user can adjust the LED brightness using a slider and can also turn the LED on or off. The last mode, motion mode, is suitable for nighttime, as the LED turns on whenever motion is detected by the motion sensor.
